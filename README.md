online-adventur.es
==================

Code for the website online-adventur.es

Prerequisites
=============
need to set a mandrill API key
* for dev export MANDRILL_API_KEY=...
* for heroku : heroku config:set MANDRILL_API_KEY=...

```bash
npm install
```

Exec
====

Prod (heroku)
```bash
foreman start
```

```bash
nodemon
```

```bash
npm start
```


```bash
git push heroku master
```
